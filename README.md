# 닥신 AI 프롬프트 생성기 (Dakshin AI Prompt Generator)

엔카(Enka) 중고차 매물 링크를 입력하면, ChatGPT나 Claude 등 대화형 AI 모델에서 사용할 수 있는 **'닥신(중고차 유튜버)' 페르소나 특화 검증 프롬프트**를 동적으로 생성해주는 웹 서비스입니다. 

## ✨ 주요 기능
- **🤖 닥신 페르소나 주입**: 가성비, 무사고, 뼈대 손상 여부, 보험이력 등을 단호하고 실용적으로 평가하는 유튜버 '닥신'의 철학과 유행어를 프롬프트에 담았습니다.
- **🌐 실시간 차량 정보 크롤링 (CORS Proxy)**: 입력한 엔카 링크를 바탕으로 퍼블릭 프록시(`allorigins.win`)를 경유해 차량명, 가격, 연식, 주행거리 등 필수 제원 정보를 실시간으로 파싱하여 프롬프트 본문에 자동으로 주입합니다. AI가 자체 웹 브라우징 툴로 링크 접속에 실패하는 상황을 방지합니다.
- **🔗 강력한 URL 유효성 검증**: `http://` 여부, 모바일 단축 링크 등 다양한 형태의 입력을 자동 정규화하여 안정성을 높였습니다.
- **🎨 모던하고 유려한 다크 UI**: 글래스모피즘(Glassmorphism) 기반 반투명 패널과 동적인 백그라운드 애니메이션을 적용하여 프리미엄 사용자 경험(UX)을 제공합니다.
- **📋 원클릭 복사 기능**: 만들어진 프롬프트를 버튼 한 번으로 시각적인 성공 피드백과 함께 클립보드에 완벽히 복사합니다.

## 🛠️ 기술 스택
- 순수 **HTML5**, **CSS3**, **Vanilla JavaScript** 
- 프레임워크나 복잡한 빌드 과정 없이, 가벼우면서도 최고의 성능과 디자인을 지향합니다.

## 🚀 사용 및 실행 방법
1. 별도의 설치나 빌드 과정 없이 `index.html` 파일을 브라우저로 열어서 바로 사용할 수 있습니다.
2. 로컬 서버(예: `python -m http.server`, `npx serve`) 위에서 실행하셔도 좋습니다. CORS 테스트 시 더 안정적으로 동작합니다.
3. 생성된 프롬프트를 복사하여 **웹 브라우징(검색) 기능이 켜져 있는** ChatGPT(GPT-4 등), Claude Opuse 등에 붙여넣기만 하면 닥신 AI의 중고차 평가를 받을 수 있습니다.
