# AI 닥신 중고차 프롬프트 생성기 프로젝트 기획서

## 1. 프로젝트 개요

사용자가 엔카 링크를 입력하면\
AI 닥신(중고차 전문가 Agent)이 평가할 수 있도록 최적화된 **프롬프트만
생성해주는 웹 서비스**

### 핵심 개념

-   AI가 직접 평가하는 것이 아니라
-   사용자가 ChatGPT / Claude / Gemini 등에 복붙해서 사용할 수 있는
-   **전문가 수준 프롬프트를 자동 생성하는 플랫폼**

------------------------------------------------------------------------

## 2. 목표

-   엔카 링크 입력
-   차량 정보 자동 파싱
-   닥신 스타일 평가 프롬프트 자동 생성
-   복사 / 수정 / 다운로드 기능 제공

------------------------------------------------------------------------

## 3. 시스템 아키텍처

User Input URL\
→ API 서버\
→ 엔카 페이지 파싱\
→ 차량 정보 JSON 변환\
→ Prompt Generator\
→ 사용자에게 프롬프트 출력

------------------------------------------------------------------------

## 4. 핵심 기능

### 4.1 링크 파싱

-   차량명
-   연식
-   주행거리
-   사고 이력
-   옵션
-   가격

### 4.2 프롬프트 자동 생성

-   닥신 스타일 전문가 톤 적용
-   한국 중고차 시장 기준 반영
-   가격 적정성 분석 요청 포함

### 4.3 커스터마이징 옵션

-   구매 목적 (출퇴근 / 리셀 / 패밀리카 등)
-   예산 범위
-   위험 감수 성향

------------------------------------------------------------------------

## 5. 기본 Prompt 템플릿

``` text
You are 중고차 전문가 AI "닥신".

다음 차량 정보를 분석해줘.

차량 URL: {LINK}

Step 1: 정보 정리
- 모델/트림
- 연식
- 주행거리
- 사고 이력
- 옵션
- 등록 가격

Step 2: 평가
1) 감가 요인
2) 가격 적정성
3) 시장 평균 대비 위치
4) 구매 추천 여부

출력 형식:

[차량 요약]
Model:
Year:
Km:
Price:

[평가]
- 강점:
- 약점:
- 리스크:

[최종 의견]
- 추천 여부
- 적정 가격 범위
```
